Sobre a Oracle

A Oracle Corporation é uma empresa multinacional americana de tecnologia, especializada em software de banco de dados, computação em nuvem, e soluções corporativas. Fundada em 1977 por Larry Ellison, Bob Miner e Ed Oates, a Oracle é uma das principais fornecedoras de tecnologias que ajudam as empresas a gerenciar e processar dados em grande escala.

Produtos da Oracle

  • Oracle Database
  • Oracle Cloud Infrastructure
  • Oracle Applications (ERP, HCM, CRM)
  • Oracle Exadata
  • Oracle Java
